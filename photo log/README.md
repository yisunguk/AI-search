# 사진대지 자동화 앱 - 빠른 시작 가이드

## 📋 사용 방법

### 1️⃣ 앱 실행
브라우저에서 `index.html` 파일을 열기:
- 파일 탐색기에서 `index.html` 더블클릭
- 또는 브라우저에 파일을 드래그앤드롭

### 2️⃣ 프로젝트 정보 입력
```
📝 프로젝트명: OO아파트 신축공사
```

### 3️⃣ 작업장 설정
1. 작업장명 입력
   ```
   예) 1601동 25층 바닥철근 검측
   ```
2. 더 많은 작업장 필요시 "작업장 추가" 클릭

### 4️⃣ 사진 업로드
**방법 1: 드래그 앤 드롭**
- 사진 파일을 업로드 영역으로 드래그

**방법 2: 클릭 업로드**
- 업로드 영역 클릭 → 파일 선택

### 5️⃣ Excel 생성
"사진대지 Excel 생성하기" 버튼 클릭

---

## 📁 지원 파일 형식
- JPG, JPEG, PNG, BMP
- 최대 10MB

## ✨ 주요 기능
- 📸 EXIF 촬영일자 자동 추출
- 📅 날짜순 자동 정렬
- 🖼️ 실시간 미리보기
- 📊 작업장별 시트 생성
- 📱 모바일 지원

## 💡 팁
- 작업장명은 Excel 시트명이 됩니다
- 여러 사진을 한번에 선택할 수 있습니다
- 사진은 촬영 날짜순으로 자동 정렬됩니다

## ⚠️ 주의사항
- 최소 1개 작업장 필요
- 작업장명과 사진이 있어야 Excel 생성 가능
